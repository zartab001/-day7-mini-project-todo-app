# Day 7 Mini Project – Todo App

A simple and clean Todo application built with React as part of Day 7 training.  
This project demonstrates core React concepts, including state management, props, component composition, event handling, and conditional rendering.

---

##  Project Description

This Todo App allows users to manage daily tasks efficiently. Users can add, complete, delete, and filter todos. The application is built with functional components and React hooks, following best practices for clean, maintainable code.

---

##  Features Implemented

- Add new todos using an input form
- Display list of todos
- Mark todos as complete/incomplete
- Delete todos
- Filter todos:
  - All
  - Active
  - Completed
- Display count of remaining tasks
- Clean and modular component structure

---

##  Component Structure

App
├── TodoForm (Add new todos)
├── TodoList (Display todo list)
│ └── TodoItem (Single todo item)
├── FilterButtons (All / Active / Completed)
└── TodoCount (Remaining tasks count)

---

##  Technologies Used

- React.js
- JavaScript (ES6+)
- React Hooks (`useState`)
- CSS (basic styling)
