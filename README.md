##  Project Description

This Todo App allows users to manage daily tasks efficiently. Users can add, complete, delete, and filter todos. The application is built with functional components and React hooks, following best practices for clean, maintainable code.

---

##  Features Implemented

- Add new todos using an input form
- Display list of todos
- Mark todos as complete/incomplete
- Delete todos
- Filter todos:
  - All
  - Active
  - Completed
- Display count of remaining tasks
- Clean and modular component structure
